This is a Perfect Web Appli­cation

  • It loads quickly, even on slow connections.
  • It is accessible to screen readers.
  • It does not load unnecessary resources.
  • It does not rely on external services for core functionality.
  • It complies with the GDPR.
  • It does not use low-contrast colors.
  • Its core functionality works in all major web browsers.
  • It does not use JavaScript to do things JavaScript is not required to do.
  • It is fully functional on mobile devices.
  • It does not reimplement browser functionality like scrolling.
  • It does not change what is on the page without user interaction.

To create your own perfect web application, start with an empty HTML file, then add the features you want. Breaking any of the features of this perfect web application should be considered a top-priority bug.
